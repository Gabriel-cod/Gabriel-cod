### 🚀 About me

I'm a **Senior Software Engineer** with 5+ years of building and operating **production, end-to-end web applications** in fast-paced, high-ownership startup environments. Currently at **Cakto**, a fintech/payments platform, where I own backend features — including Python/Django/DRF services and documentation that consume them, PostgreSQL query optimization, and AWS/Kubernetes/Docker deployments.

Some things I've shipped that I'm glad to have worked on since the start of my career:

- 📉 Cut a high-volume PostgreSQL query's **cost by 98%** and **execution time by ~45%** via Django ORM optimization (killed N+1 queries, tuned indexes).
- 🧮 Built a lightweight math-based product categorization model — **80% accuracy with zero AI API cost** (Python, vectorization).
- 🔒 Led an integration system that could **prevent ~$300K in fraud** by connecting financial, production, and engineering data via webhooks.
- ⚡ Designed data-driven APIs that collapsed a dashboard's frontend calls **down to a single request**, serving 20,000+ concurrent users.
- 🔑 Built a password-less (magic-link) auth flow that **cut login time by ~50%**, with throttling, IP security, and full test coverage.
- 🪙 Built a multi-tenant platform monitoring **2.4M+ cryptocurrencies**, scaled for 30,000+ simultaneous requests on AWS EC2.
